UNIVERSE DESCRIPTION
The universe is a sample of manufacturing establishments
classified in sectors 31-33 with one or more paid employees at any
time during the year.
DATA ITEMS
This file contains data on:

  Total employment
  Relative standard error of total employment (%)
  Average annual production workers
  Total annual payroll ($1,000)
  Production workers wages ($1,000)
  Production workers hours (1,000)
  Total cost of materials ($1,000)
  Value added ($1,000)
  Relative standard error of value added (%)
  Total value of shipments ($1,000)
  Total capital expenditures (new and used) ($1,000)
  Relative standard error of total capital expenditures (new
  and used) (%)

GEOGRAPHIC COVERAGE
Data are shown at the U.S. and state levels.
INDUSTRY DETAIL
Data are shown at the three- and four-digit North American
Industry Classification System (NAICS) levels.
OTHER VARIABLES IDENTIFYING RECORDS
None.
RELEASE SCHEDULE
Data are scheduled for release in December 2010.
RELATED DATA FILES

DATA LIMITATIONS
Data are subject to sampling and non-sampling errors.
FILE SEQUENCE
Data are presented in ascending geography (U.S. x ST) by NAICS
code sequence.
