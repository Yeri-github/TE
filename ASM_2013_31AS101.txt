AM1331AS101
Annual Survey of Manufactures: Geographic Area Statistics: Statistics for All Manufacturing by State: 2013

Release Date : 02/27/2015

NOTE: For information on confidentiality protection, sampling error, nonsampling error, and definitions, see Survey Methodology. Data in this table represent those available when this report was created. Data may not be available for all NAICS industries or geographies. Detail items may not add to total due to independent rounding.  For information on ASM, see the Annual Survey of Manufactures (ASM) Website.

 
  
  Table Name 
  Annual Survey of Manufactures: Geographic Area Series: Detailed Statistics for All Manufacturing by State: 2013 
  
  
  ReleaseSchedule 
  Data are scheduled for release in February 2015. 
  
  
  Universe 
  The universe is a sample of manufacturing establishments classified in sectors 31-33 with one or more paid employees at any time during the year. 
  
  
  GeographyCoverage 
  Data are shown at the U.S. and state levels. 
  
  
  IndustryCoverage 
  Data are shown at the two-, three-, and four-digit North American Industry Classification System (NAICS) levels. 
  
  
  Data ItemsandOtherIdentifyingRecords 
  This file contains data on: 
    
    Number of employees 
    Relative standard error for estimate of number of employees (%) 
    Annual payroll ($1,000) 
    Relative standard error for estimate of annual payroll (%) 
    Production workers average for year 
    Relative standard error for estimate of production workers average for year (%) 
    Production workers annual hours (1,000) 
    Relative standard error for estimate of production workers annual hours (%) 
    Production workers annual wages ($1,000) 
    Relative standard error for estimate of production workers annual wages (%) 
    Total cost of materials ($1,000) 
    Relative standard error for estimate of total cost of materials (%) 
    Total value of shipments and receipts for services ($1,000) 
    Relative standard error for estimate of total value of shipments and receipts for services (%) 
    Value added ($1,000) 
    Relative standard error for estimate of value added (%) 
    Total capital expenditures ($1,000) 
    Relative standard error for estimate of capital expenditures (%) 
     
  
  
  Sort Order 
  Data are presented in ascending state code by NAICS code sequence.
